How to use:
-----

    python AES.py -demo
    python AES.py (-e | -d) <file> [-c (128|192|256)]
        -e: Encript
        -d: Decript
        -c <n>: <n> bits key (default 128)
    Note: a function mode (-e/-d) has to be specified.
