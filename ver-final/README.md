Como usar:
-----
    python AES.py -demo
    python AES.py (-e | -d | -ce | -cd) <file>
        -e: Encrypt (Criptografar)
        -d: Decrypt (Decriptografar)
        -ce: Calcula o tempo de criptografia
        -cd: Calcula o tempo de descriptografia
    Note: um modo de função (-e/-d/-ce/-cd) deve ser especificado.
